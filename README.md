3Dアニメーション付きの銃を追加します。マスケット、小銃、短機関銃

・フリントロック式マスケット銃

・ボルトアクション式連発銃(Gewehr 98)

・初期型短機関銃(Beretta M1918)

こだわり

・マスケット銃はBキーで着剣が可能、着剣時に右クリックで刺突ができます。リロードが10秒くらいかかる上に単発なので必須ですね

・マスケット銃を撃つと大きな煙が発生します。

導入方法

・jarファイルをダウンロードして、modsフォルダににぶち込んでください。

・前提modとしてGeckolibが必須です

・現状1.20.1でしか動きません。

・今のところクリエイティブインベントリにアイテムが表示されないのでコマンドで出すか作ってください。

/give @p ygunmod:flintlock_musket

/give @p ygunmod:ballbullet 64

↑例

需要

・塹壕戦ごっこしたい人

・戦列歩兵ごっこしたい人
